欢迎使用 ShareCraft

        
把当前网页一键变成可直接转发的推荐语。下面快速看一下怎么用。

      


      

        
① 配置你自己的 API Key

        
插件用你自己的 LLM Key 调用，支持 OpenAI 和 Anthropic 协议（含兼容网关）。

        
点右上角 ⚙ 进入设置页，填好 Key 和 Base URL，点「测试连接」。

      


      

        
② 选模板，一键生成

        
勾选你想要的分享模板（朋友安利 / 金句 / 详细，或你自定义的），点「生成推荐语」。

        
每个模板会并发独立请求，更快、互不影响；可以随时点「终止」。

      


      

        
③ 复制 → 粘贴到聊天

        
每条推荐语下方有「复制」按钮，会自动拼接「推荐语 + 网页链接」，直接粘到微信 / 工作群即可。

        
关掉窗口任务也会继续；历史结果可以在 🕓 里翻出来。
